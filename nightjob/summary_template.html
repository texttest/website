Nightjob results for TextTest and StoryText
The above table shows the current status of the self-tests. Currently these tests are run every night using GTK+ 2.20 on Red Hat Enterprise Linux 6 and also on Windows 7.
Naturally we make every effort to ensure that these are always green, but there are currently a small number that contain race conditions and fail about one time in ten. By following the links above, you can check in the nightjob history to see how frequently a test has failed in the past and hence how indeterministic it is.
We've discontinued measuring coverage for the moment, as it seemed to be more effort than it was worth at this mature stage of the product.
