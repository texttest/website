Welcome to the home of Texttest
Texttest is an application-independent Open Source tool for text-based functional testing. This means running a batch-mode binary in lots of different ways, and using the text output produced as a means of controlling the behaviour of that application.
The developemnt base for Texttest currently resides on SourceForge
